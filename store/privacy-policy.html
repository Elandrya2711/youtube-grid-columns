Privacy Policy

Last updated: February 2026  ·  YouTube Grid Columns

Overview

YouTube Grid Columns is a browser extension that lets you control the number of video columns on YouTube's home page. This policy explains what data the extension accesses and how it is handled.

Data We Collect

The extension stores a single value: your chosen number of columns (a number between 1 and 12). This value is saved using Chrome's built-in chrome.storage.sync API, which syncs it across your devices through your Google account.

We do not collect, transmit, or have access to any of this data. The value never leaves Chrome's own storage system and is never sent to any external server.

Data We Do Not Collect

  • No personal information (name, email, IP address, etc.)
  • No browsing history or watch history
  • No analytics or usage data
  • No cookies

Permissions

The extension requests the following permissions:

  • storage — to save and sync your column preference across devices via your Google account.
  • Host permission for youtube.com — to inject the CSS that adjusts the grid layout on YouTube pages. The extension reads no data from YouTube and makes no network requests.

Third Parties

The extension does not use any third-party services, analytics tools, or advertising networks. No data is shared with anyone.

Changes to This Policy

If the extension is updated in a way that changes how data is handled, this page will be updated accordingly.

Contact

If you have questions about this privacy policy, you can open an issue in the extension's repository or reach out via the Chrome Web Store listing.
